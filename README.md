# What Even, BE (Productive)
A JavaScript web extension, using Google Gemini.
---
## Features
- User set baeline for amount of scrolls allowed
- Interupts all scrolling during user selected bedtime
- Task planning system with deadline tracker and progress bar
- AI interuption profiles for personalized messages when scrolling is interupted
- User selected interuption rate
---
## Install
### 1. Download Repository
  As a ZIP from github, then extract or copy
### 2. Add API key
  open AI.Js in any editor and insert API key from Google AI Studio 
### 3. Add to extensions
  - click the "extensions" button on chrome
  - click "Manage Extensions"
  - select "Developer mode"
  - Click "Load Unpacked" and insert extension-Main folder
